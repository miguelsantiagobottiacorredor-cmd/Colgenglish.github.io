Actividad 1

instrucciones

El estudiante debe leer cuidadosamente cada pregunta y seleccionar la opción correcta entre las cuatro alternativas disponibles (A, B, C y D). Para responder adecuadamente, debe analizar el enunciado y aplicar los conocimientos adquiridos durante la lección. Solo una de las opciones es la correcta. Esta actividad le permitirá evaluar su comprensión del tema trabajado y reforzar su dominio del idioma inglés.

Pregunta de Elección Múltiple

Pregunta

1. This house is ____ than the one we saw yesterday.

Respuestas

more big

bigger

the biggest

most big

Retroalimentación

incorrecto

“Big” es un adjetivo corto. No se usa “more” con adjetivos de una sílaba.

correcto

“Big” se convierte en “bigger” al formar el comparativo. Solo se añade “-er”.

 

incorrecto

Esta es la forma superlativa. Aquí estamos comparando dos casas, así que se necesita el comparativo.

incorecto

Esta forma no existe en inglés. Se debe usar “bigger”.

Solución

  1. Incorrecto (Retroalimentación)
  2. Opción correcta (Retroalimentación)
  3. Incorrecto (Retroalimentación)
  4. Incorrecto (Retroalimentación)

Pregunta

2. She is the ____ student in the class.

Respuestas

smarter

more smart

Smart

most smarter

Retroalimentación

incorrecto

“Smarter” es comparativo, no superlativo. Aquí hablamos del grado más alto dentro del grupo.

incorrecto

“Smart” es un adjetivo corto, se usa “-er” o “-est”, no “more”.

correcto

“Smart” + “-est” = “smartest”, y se usa “the” con superlativos.

incorercto

No se puede usar “most” con un comparativo como “smarter”. Es redundante e incorrecto.

Solución

  1. Incorrecto (Retroalimentación)
  2. Incorrecto (Retroalimentación)
  3. Opción correcta (Retroalimentación)
  4. Incorrecto (Retroalimentación)

Pregunta

3.My dog is ____ than your dog.

Respuestas

friendlier

more friendlier

most friendly

friendly

Retroalimentación

correcto

Aunque “friendly” termina en “-ly”, se acepta como adjetivo regular: “friendlier”.

incorrecto

No se usa “more” con una forma ya comparativa como “friendlier”.

incorrecto

“Most” se usa en superlativos, no en comparaciones entre dos elementos.

incorrecto

Es solo el adjetivo base. Falta la comparación.

Solución

  1. Opción correcta (Retroalimentación)
  2. Incorrecto (Retroalimentación)
  3. Incorrecto (Retroalimentación)
  4. Incorrecto (Retroalimentación)

Pregunta

4.That was the ____ movie I’ve ever seen.

Respuestas

most exciting

more exciting

excitingest

most excited

Retroalimentación

correcto

“Exciting” es un adjetivo largo. Usamos “most” para formar el superlativo.

incorrecto

“More exciting” es comparativo. Aquí se busca la forma superlativa.

incorrecto

No existe esta forma. Los adjetivos largos no llevan “-est”.

incorrecto

“Excited” describe a una persona, no una película. Aquí se necesita “exciting”.

Solución

  1. Opción correcta (Retroalimentación)
  2. Incorrecto (Retroalimentación)
  3. Incorrecto (Retroalimentación)
  4. Incorrecto (Retroalimentación)

Pregunta

5.John is ____ than Mike.

Respuestas

tall

the taller

tallest

taller

Retroalimentación

incorrecto

Esta es la forma base del adjetivo, pero se necesita una comparación.

incorrecto

“The” no se usa con comparativos como “taller”.

incorrecto

Esta es la forma superlativa, usada cuando se comparan más de dos.

correcto

“Taller” es el comparativo adecuado para dos personas.

Solución

  1. Incorrecto (Retroalimentación)
  2. Incorrecto (Retroalimentación)
  3. Incorrecto (Retroalimentación)
  4. Opción correcta (Retroalimentación)

Pregunta

6.This is the ____ cake I’ve ever tasted!

Respuestas

gooder

more good

best

most better

Retroalimentación

incorrecto

“Good” es irregular. No se le añade “-er”.

incorrecto

Tampoco se usa “more” con “good”. Su forma especial es “best”.

correcto

“Good” → “best” en el superlativo. Es una excepción.

incorrecto

“Better” ya es comparativo. “Most” no debe combinarse con él.

Solución

  1. Incorrecto (Retroalimentación)
  2. Incorrecto (Retroalimentación)
  3. Opción correcta (Retroalimentación)
  4. Incorrecto (Retroalimentación)

Pregunta

7.Today is ____ than yesterday.

Respuestas

more hot

hotter

hottest

the hotter

Retroalimentación

incorrecto

“Hot” es un adjetivo corto. Se usa “-er”, no “more”.

correcto

Duplicamos la “t” porque “hot” termina en consonante-vocal-consonante.

incorrecto

Es superlativo, y aquí solo comparamos dos días.

 incorrecto

El artículo “the” no se usa con comparativos.

Solución

  1. Incorrecto (Retroalimentación)
  2. Opción correcta (Retroalimentación)
  3. Incorrecto (Retroalimentación)
  4. Incorrecto (Retroalimentación)

Pregunta

8.That mountain is the ____ in the region.

Respuestas

higher

 more high

highest

most high

Retroalimentación

incorrecto

Esta forma es comparativa, no superlativa.

incorrecto

“High” es corto, así que no se usa “more” con él.

correcto

“High” → “highest” para expresar el grado más alto dentro de un grupo.

incorrecto

“Most high” no es una forma correcta de superlativo para este caso.

Solución

  1. Incorrecto (Retroalimentación)
  2. Incorrecto (Retroalimentación)
  3. Opción correcta (Retroalimentación)
  4. Incorrecto (Retroalimentación)
